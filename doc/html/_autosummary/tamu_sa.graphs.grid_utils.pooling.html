tamu_sa.graphs.grid_utils.pooling¶

tamu_sa.graphs.grid_utils.pooling(mat, ksize, method='max', pad=False)[source]¶

Non-overlapping pooling on 2D or 3D data.

<mat>: ndarray, input array to pool. <ksize>: tuple of 2, kernel size in (ky, kx). <method>: str, ‘max for max-pooling,

‘mean’ for mean-pooling.

<pad>: bool, pad <mat> or not. If no pad, output has size

n//f, n being <mat> size, f being kernel size. if pad, output has size ceil(n/f).

Return <result>: pooled matrix.
